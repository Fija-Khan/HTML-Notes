🎨 CSS Colors + Background + Units

🎨 1. CSS COLORS (Full Explanation)
Definition: CSS colors are used to define color of text, background, borders, etc.
Simple Meaning: Colors make website attractive and readable.
Why Colors used?
• To highlight content
• To improve design
• To create good UI
Types of CSS Colors (Detailed):

1. Named Colors:
These are simple predefined color names given by CSS.
Easy to use but limited options.
Example:
color: red;
color: blue;
color: green;

Explanation: You just write color name directly. Browser already understands it.

2. HEX Colors:
HEX means Hexadecimal color code. It starts with # symbol.
It is most commonly used in professional websites.
Example:
color: #ff0000; (red)
color: #000000; (black)
color: #ffffff; (white)

Explanation: It uses Red, Green, Blue values in hexadecimal format (0–9, A–F).

3. RGB Colors:
RGB stands for Red, Green, Blue.
Each value ranges from 0 to 255.
Example:
color: rgb(255, 0, 0); → red
color: rgb(0, 255, 0); → green
color: rgb(0, 0, 255); → blue

Explanation: Different colors are created by mixing red, green, blue light.

4. RGBA Colors:
RGBA = RGB + Alpha (opacity / transparency)
Example:
color: rgba(255, 0, 0, 0.5);

Explanation: Last value (0 to 1) controls transparency:
0 = fully transparent
1 = fully visible

5. HSL Colors:
HSL stands for Hue, Saturation, Lightness.
Example:
color: hsl(0, 100%, 50%);

Explanation:
Hue → color type (0–360)
Saturation → color intensity
Lightness → brightness of color
🖼️ 2. CSS BACKGROUND (Full Explanation)
Definition: Background is used to set color or image behind an element.
Types:
• Background Color
• Background Image
• Gradient Background
Background Color:
background-color: yellow;
Background Image:
background-image: url("image.jpg");
Important Properties:
• cover → full screen fit
• center → image center
• no-repeat → stop repeat
Gradient Example:
linear-gradient(red, yellow);
💡 Real Life Example: Like mobile wallpaper or room paint.
🎯 Interview Question: Difference between cover and contain?
📏 3. CSS UNITS (Full Explanation)
Definition: CSS units define size of elements like width, height, font-size.
Types:
1. px: Fixed size unit, most commonly used.
2. %: Depends on parent size.
3. em: Depends on parent font size.
4. rem: Depends on root font size.
5. vh: Viewport height.
6. vw: Viewport width.
Example:
font-size: 20px;
width: 50%;
height: 100vh;
💡 Real Life Example: Like measuring clothes in different sizes.
🎯 Interview Question: Difference between px, em, rem?
🚀 4. FINAL SUMMARY
✔ Colors → Design beauty
✔ Background → Visual look
✔ Units → Size control
✔ These are CSS foundation concepts
🔥 Final Line: CSS is used to make websites modern, attractive and responsive.
