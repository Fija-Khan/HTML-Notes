📦 CSS BOX MODEL

Learn → Understand → Practice → Test Yourself
1. WHAT IS BOX MODEL?
Definition: Every HTML element is a rectangular box that has content, padding, border, and margin.
📌 Real Life Example: Gift Box 🎁 = Content (gift) + Padding (foam) + Border (box) + Margin (space)
2. CONTENT
Meaning: The actual text, image or data inside the element.
div { width: 200px; height: 100px; }
💡 Example: Text inside a button is content.
3. PADDING
Meaning: Space between content and border.
div { padding: 20px; }
💡 Example: Cushion inside a box protecting items.
4. BORDER
Meaning: Line around the element.
div { border: 2px solid black; }
💡 Example: Outer cover of a gift box.
5. MARGIN
Meaning: Space outside the element.
div { margin: 30px; }
💡 Example: Distance between two boxes on a table.
