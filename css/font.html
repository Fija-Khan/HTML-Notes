✍️ CSS FONT & TEXT STYLING

Learn with real life examples + simple explanations + practice
1. FONT FAMILY
Meaning: Font-family is used to change the style of text (like handwriting style).
Real Life Example: Just like different people have different handwriting styles, websites also have different font styles.
Example: Newspaper uses simple font, while Instagram uses modern stylish font.
p { font-family: Arial, Helvetica, sans-serif; }
2. FONT SIZE
Meaning: Controls how big or small text appears.
Real Life Example: Headings in books are big, normal text is small.
h1 { font-size: 30px; }
3. FONT WEIGHT
Meaning: Makes text bold or thin.
Real Life Example: Important words in books are bold so we notice them easily.
p { font-weight: bold; }
4. TEXT ALIGN
Meaning: Aligns text (left, right, center).
Real Life Example: School notices are always center aligned for better visibility.
h1 { text-align: center; }
5. TEXT TRANSFORM
Meaning: Changes text into uppercase or lowercase.
Real Life Example: Headlines in newspapers are always in CAPITAL letters.
p { text-transform: uppercase; }
6. TEXT DECORATION
Meaning: Adds or removes underline.
Real Life Example: Links on websites are underlined like important references.
a { text-decoration: none; }
7. LINE HEIGHT
Meaning: Space between lines of text.
Real Life Example: If lines are too close, reading becomes difficult like crowded writing.
p { line-height: 1.6; }
