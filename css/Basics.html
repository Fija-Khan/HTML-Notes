CSS

Deep Understanding of CSS & Selectors

1. What is CSS?

CSS stands for Cascading Style Sheets.

CSS is used to design, style and decorate HTML webpages.

HTML gives structure (skeleton) CSS gives design (skin + beauty)

Without CSS, websites look like plain text pages.

CSS controls:

  • Colors
  • Layout
  • Spacing
  • Fonts
  • Animations

2. Why CSS is Used?

  • To make website attractive
  • To improve user experience
  • To control layout structure
  • To make website responsive
  • To separate design from HTML

Example: Instagram, YouTube, Amazon all use CSS for design

3. Types of CSS

Inline CSS

CSS written inside HTML tag using style attribute.

<h1 style="color:red;">Hello</h1>

Not reusable and not good for large projects.

Internal CSS

CSS written inside style tag in HTML file.

<style> h1 { color: blue; } </style>

Used for single page websites.

External CSS

CSS written in separate .css file.

Best and professional method.

  • Reusable
  • Clean code
  • Easy maintenance
style.css file: h1 { color: green; }

4. What is CSS Selector?

Selector is used to select HTML elements and apply CSS styles.

CSS needs selectors to know which element to style.

Element Selector

Selects all elements of same type.

p { color: white; }

Universal Selector

Selects all elements in webpage.

* { margin: 0; padding: 0; }

Class Selector

Used for multiple elements. Starts with dot (.)

  • Reusable
  • Same style for multiple elements
.box { color: blue; }

ID Selector

Used for single unique element. Starts with #

  • Unique styling
  • Single element only
#header { color: red; }

Group Selector

Used to apply same style to multiple elements.

h1, h2, p { color: green; }

Final Summary

  • CSS is used for design
  • Three types: Inline, Internal, External
  • Selectors are used to target elements
  • Class is reusable
  • ID is unique
  • Universal targets all elements
