📘 HTML Day 2 Notes

1. HTML Structure

Every HTML page follows a fixed structure using basic tags.

<!DOCTYPE html>
<html>
<head>
<title>My First Website</title>
</head>
<body>
Content here
</body>
</html>

2. Explanation of Structure

  • DOCTYPE → Defines HTML5 version
  • html → Root element
  • head → Contains metadata (not visible)
  • title → Shows browser tab name
  • body → Visible content of webpage

3. First Website Example

<!DOCTYPE html>
<html>
<head>
<title>My Website</title>
</head>
<body>
<h1>Welcome to My Website 🚀</h1>
<p>This is my first HTML page</p>
</body>
</html>

4. Real Life Example

  • HTML = Structure 🏗
  • CSS = Design 🎨
  • JavaScript = Function ⚙️

5. Important Points

  • Always start with DOCTYPE
  • Head is not visible
  • Body is visible part
  • Structure is base of every website

Interview Questions

Q1: What is HTML structure?

HTML structure defines the basic layout using html, head, and body tags.

Q2: What is DOCTYPE?

It tells browser we are using HTML5 version.

Q3: What is head tag?

It contains meta information like title and links.

Q4: What is body tag?

It contains visible content of webpage.

Q5: Difference between head and body?

Head = setup information, Body = visible content
