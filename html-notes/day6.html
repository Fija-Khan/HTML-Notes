HTML Day 6

Forms and Input Elements

1. What is HTML Form?

An HTML form is used to collect user input. It is used in login pages, signup pages, contact forms, and many real-world applications.

Forms help us send data from user to server.

2. Basic Form Structure

<form> form elements here </form>

3. Input Tag

The input tag is used to take user data in different formats.

It is a self-closing tag.

<input type="text">

4. Types of Input Fields

Text

<input type="text">

Email

<input type="email">

Password

<input type="password">

Number

<input type="number">

Radio

<input type="radio">

Checkbox

<input type="checkbox">

File Upload

<input type="file">

5. Other Form Elements

Textarea

<textarea></textarea>

Select Dropdown

<select> <option>Option 1</option> </select>

Button

<button>Submit</button>

6. Form Attributes

  • action - where data is sent
  • method - GET or POST
<form action="server.php" method="post">

7. HTML Validation

  • required - field must be filled
  • type="email" - checks email format
  • min / max - limit values
  • pattern - custom validation
<input type="email" required>

8.Form Example

<form> <label>Name</label> <input type="text" required> <label>Email</label> <input type="email" required> <label>Password</label> <input type="password" required> <label>Gender</label> <input type="radio" name="g"> Male <input type="radio" name="g"> Female <label>Skills</label> <input type="checkbox"> HTML <input type="checkbox"> CSS <label>Message</label> <textarea></textarea> <button>Submit</button> </form>

9. Important Points

  • Forms collect user data
  • Use proper input types
  • Always use validation
  • Used in all real websites

10. Common Mistakes

  • Missing name attribute
  • Not using label
  • Wrong input type
  • No validation
