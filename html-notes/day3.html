📘 HTML Day 3 - Registration Form

1. What is HTML Form?

An HTML form is used to collect user input such as name, email, password, etc.

Forms are very important in real-world websites like login pages, signup pages, and surveys.

2. Form Structure

<form> form elements here </form>

3. Input Fields

  • text → Name input
  • email → Email input
  • password → Password input
  • number → Phone number
  • file → File upload

4. Full Registration Form Example

<form> <label>Name:</label>
<input type="text" required>

<label>Email:</label>
<input type="email" required>

<label>Phone:</label>
<input type="number">

<label>Password:</label>
<input type="password" required>

<label>Upload File:</label>
<input type="file">

<button type="submit">Submit</button> </form>

5. HTML Validation

  • required → field must be filled
  • type="email" → checks email format
  • pattern → custom validation rule

6. Real World Use

  • Login pages
  • Signup forms
  • Contact forms
  • Online applications

7. Important Points

  • Forms collect user data
  • Always use validation
  • Use proper input types
  • Forms are used in all websites

🎯 Interview Questions

Q1: What is HTML form?

It is used to collect user input like name, email, etc.

Q2: What is input tag?

It is used to take user input in different formats.

Q3: What is required attribute?

It makes input field mandatory.

Q4: Why email type is used?

It validates email format automatically.

Q5: Where are forms used?
