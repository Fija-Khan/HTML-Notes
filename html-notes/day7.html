Day7- HTML5 Structure

1. What is HTML?

HTML (HyperText Markup Language) is used to create the structure of a webpage.

It defines elements like headings, paragraphs, images, and links.

2. What is HTML5?

HTML5 is the latest version of HTML.

It introduces new tags and features that make websites more modern, structured, and user-friendly.

3. Difference Between HTML and HTML5

  • HTML is older version, HTML5 is latest
  • HTML5 supports audio, video, and new elements
  • HTML5 provides better structure using semantic tags
  • HTML5 is more SEO friendly

4. What are Semantic Tags?

Semantic tags clearly describe the meaning of the content.

They help browsers, developers, and search engines understand the structure of a webpage.

5. HTML5 Structure Tags

Header

Represents the top section (logo, title, navigation).

Nav

Used for navigation links (menus).

Section

Defines a section of content.

Article

Represents independent content (blog, post).

Footer

Bottom part of the webpage (copyright, links).

6. Basic HTML5 Structure

<header> Website Header </header> <nav> Home | About | Contact </nav> <section> <article> This is a blog post </article> </section> <footer> Footer content </footer>

7. Real-Life Example (E-commerce Website)

In an online shopping website:

  • Header → Logo + Search Bar
  • Nav → Categories (Men, Women, Electronics)
  • Section → Product sections
  • Article → Single product details
  • Footer → Contact, links, policies

8. Complete Example

<header> My Website </header> <nav> Home | About | Contact </nav> <section> <article> Welcome to my website </article> </section> <footer> All rights reserved </footer>

9. Important Points

  • HTML creates structure
  • HTML5 adds modern features
  • Semantic tags improve readability
  • Helps SEO and accessibility

10. Practice Tasks

  • Create basic layout using header, nav, footer
  • Add section and article
  • Build simple e-commerce layout
