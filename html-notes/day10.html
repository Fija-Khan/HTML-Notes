SEO Basics in HTML

1. What is SEO?

SEO stands for Search Engine Optimization. It helps your website appear on Google search results.

👉 Simple meaning: SEO makes your website visible on search engines.

2. Why SEO is Important?

  • Increases website traffic
  • Improves Google ranking
  • Makes website discoverable
  • Important for business websites

3. Title Tag

Title tag defines the name of your webpage. It appears in browser tab and Google search.

<title>Learn HTML Step by Step</title>

👉 Always use meaningful title for SEO.

4. Meta Description

Meta description gives short summary of webpage. It appears in Google search results.

<meta name="description" content="Learn HTML basics with examples">

5. Viewport Tag

This tag makes website mobile friendly.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

6. Semantic HTML Tags

Semantic tags give meaning to webpage structure.

<header>Header Section</header> <nav>Navigation</nav> <section>Main Content</section> <article>Article Content</article> <footer>Footer</footer>

👉 These tags help Google understand your content.

7. Complete SEO Example

<!DOCTYPE html> <html> <head> <title>Learn HTML SEO</title> <meta name="description" content="HTML SEO basics tutorial"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> </head> <body> <header> <h1>Welcome to HTML SEO</h1> </header> <nav> Home | About | Contact </nav> <section> <p>This is SEO optimized page</p> </section> <footer> <p>Copyright 2026</p> </footer> </body> </html>
