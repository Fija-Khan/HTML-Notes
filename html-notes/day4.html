HTML Day 4

Images, Links and Lists

What You Will Learn

  • How to add images in a webpage
  • How to create clickable links
  • Different types of lists in HTML

1. Image Tag (img)

The image tag is used to display images on a webpage. It makes the website visually attractive and meaningful.

It is a self-closing tag.

<img src="image.jpg" alt="My Image" width="200">

Attributes:

  • src - Image path
  • alt - Alternative text
  • width - Image size

Always use alt attribute for better accessibility.

2. Anchor Tag (a)

Anchor tag is used to create links between pages or websites.

<a href="https://google.com">Visit Google</a>

Attribute:

  • href - Link destination

Open link in new tab:

<a href="https://google.com" target="_blank">Open Google</a>

3. Lists in HTML

HTML lists are used to organize content in a structured and readable format.

List Item Tag (li)

The li tag stands for List Item. It is used to define each item inside a list.

The li tag is always used inside unordered list (ul) or ordered list (ol). It cannot be used alone.

Why we use li:

  • To organize content clearly
  • To create lists of items
  • To show steps or structured data

Important Points:

  • li must be inside ul or ol
  • Each li represents one item
  • You can use multiple li tags

Types of Lists:

  • Unordered List (ul)
  • Ordered List (ol)
  • Description List (dl)

Unordered List (ul)

Used when order does not matter. Shows bullet points.

<ul> <li>HTML</li> <li>CSS</li> </ul>

Types of Unordered List (ul)

Unordered list uses different bullet styles using CSS.

<ul style="list-style-type: disc;"> <li>HTML</li> </ul> <ul style="list-style-type: circle;"> <li>HTML</li> </ul> <ul style="list-style-type: square;"> <li>HTML</li> </ul> <ul style="list-style-type: none;"> <li>HTML</li> </ul>

Ordered List (ol)

Used when order is important. Shows numbering.

<ol> <li>Step 1</li> <li>Step 2</li> </ol>

Types of Ordered List (ol)

<ol type="1"> <li>Item</li> </ol> <ol type="A"> <li>Item</li> </ol> <ol type="a"> <li>Item</li> </ol> <ol type="I"> <li>Item</li> </ol> <ol type="i"> <li>Item</li> </ol>

Description Data Tag (dd)

The dd tag is used to define the description of a term.

<dl> <dt>HTML</dt> <dd>Structure of webpage</dd> </dl>
