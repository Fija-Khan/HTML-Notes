Tables in HTML

1. What is a Table?

An HTML table is used to display data in a structured format using rows and columns. It helps to organize information clearly and makes it easy to read.

Tables are commonly used in real-world websites like reports, dashboards, and student data systems.

2. Table Structure

HTML table is created using multiple tags:

  • table - main container of the table
  • tr - table row
  • th - table heading (bold and centered)
  • td - table data (normal content)

Important Concept:

Table = Rows + Columns

<table> <tr> <th>Heading</th> </tr> <tr> <td>Data</td> </tr> </table>

3. Basic Table Example

<table border="1"> <tr> <th>Name</th> <th>Age</th> </tr> <tr> <td>Ali</td> <td>20</td> </tr> <tr> <td>John</td> <td>22</td> </tr> </table>
Name Age
Ali 20
John 22

5. Table Attributes

Border

<table border="1">

Cell Padding

<table cellpadding="10">

Cell Spacing

<table cellspacing="10">

6. Colspan

Used to merge multiple columns into one.

<th colspan="2">Student Info</th>

7. Rowspan

Used to merge multiple rows into one.

<th rowspan="2">Name</th>

8. Advanced Table Tags

  • thead - table header section
  • tbody - table body section
  • tfoot - table footer section
<table> <thead> <tr> <th>Name</th> </tr> </thead> <tbody> <tr> <td>Ali</td> </tr> </tbody> </table>

9. Important Points

  • Tables are used to display structured data
  • Use th for headings and td for data
  • Always maintain proper structure
  • Use colspan and rowspan carefully

10. Common Mistakes

  • Forgetting tr tag
  • Wrong colspan or rowspan values
  • Not closing tags properly
  • Using td instead of th for headings
