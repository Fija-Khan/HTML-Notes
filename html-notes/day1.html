📘 HTML Day 1 Notes

1. What is HTML?

HTML stands for HyperText Markup Language.

It is used to create the structure of web pages.

HTML is not a programming language, it is a markup language.

2. Who Created HTML?

HTML was created by Tim Berners-Lee in 1991.

He is also known as the father of the World Wide Web 🌐.

3. More Information About HTML

  • HTML uses tags to create elements
  • Most tags have opening and closing tags
  • Example: <p>Hello</p>
  • HTML works with CSS and JavaScript

4. Real Life Example

  • HTML = Structure 🏗
  • CSS = Design 🎨
  • JavaScript = Function ⚙️

5. Basic Structure of HTML

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
Hello World
</body>
</html>

Explanation:

  • <!DOCTYPE html> → Defines HTML5
  • <html> → Root element
  • <head> → Contains meta info
  • <body> → Visible content

6. What are Tags?

Tags are used to define elements in HTML.

Examples: <h1>, <p>, <a>
