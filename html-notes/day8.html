Media in HTML

iframe, video and audio tags

1. What is Media in HTML?

Media means adding video, audio, and external content into a webpage.

It makes your website more interactive and user-friendly.

2. iframe Tag

The iframe tag is used to embed external content like YouTube videos or websites.

<iframe src="https://www.youtube.com/embed/VIDEO_ID"></iframe>

Important Points

  • Always use embed link
  • Set width and height
  • Use allowfullscreen

Common Error

  • Using normal YouTube link
  • Wrong video ID

3. Video Tag

The video tag is used to add videos from your system or server.

<video width="400" controls> <source src="video.mp4" type="video/mp4"> </video>

Attributes

  • controls - show buttons
  • autoplay - start automatically
  • loop - repeat video
  • muted - mute sound

4. Audio Tag

The audio tag is used to add sound or music.

<audio controls> <source src="audio.mp3" type="audio/mpeg"> </audio>

5. When to Use What

  • Use iframe for YouTube or external content
  • Use video for your own video files
  • Use audio for sound files

6. Supported Formats

  • Video: mp4, webm
  • Audio: mp3, wav

7. Complete Example

<iframe width="300" height="200" src="https://www.youtube.com/embed/VIDEO_ID"></iframe> <video width="300" controls> <source src="video.mp4" type="video/mp4"> </video> <audio controls> <source src="audio.mp3" type="audio/mpeg"> </audio>

9. Important Points

  • iframe is used for external content
  • video tag is for local files
  • audio tag is for sound
  • Always use controls
  • Use correct file path

10. Common Mistakes

  • Wrong YouTube link
  • Missing controls attribute
  • Wrong file path
  • Unsupported format
